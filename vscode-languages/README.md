# VS Code Built-in Languages

This extension provides TextMate grammars extracted from VS Code's built-in language extensions for syntax highlighting in Faraday.

## Included Languages

- **C/C++**: c, cpp, cuda-cpp
- **JavaScript/TypeScript**: javascript, javascriptreact, typescript, typescriptreact
- **Python**: python
- **Rust**: rust
- **Go**: go
- **Java**: java
- **C#**: csharp
- **Ruby**: ruby
- **PHP**: php
- **Shell**: shellscript, powershell
- **Web**: html, css, scss, less, json, xml, yaml
- **And many more**: clojure, coffeescript, dart, docker, fsharp, groovy, handlebars, ini, julia, latex, lua, make, markdown, objective-c, perl, pug, r, razor, restructuredtext, sql, swift, vb

## Total

- **71 languages** with file associations
- **81 TextMate grammars** for syntax highlighting
